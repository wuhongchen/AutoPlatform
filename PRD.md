## 1. 产品概述

### 1.1 产品定位

AutoPlatform 是一个**全自动文章运营平台**，面向内容创作者、自媒体运营者，提供从文章采集、AI改写、排版到发布的一站式解决方案。

### 1.2 目标用户

| 用户类型 | 特征 | 需求 |
|---------|------|------|
| 自媒体运营者 | 管理1-3个公众号，日更压力大 | 自动化内容生产，降低人力成本 |
| 内容团队 | 多账号矩阵运营 | 批量处理，风格统一 |
| AI内容创作者 | 探索AI辅助创作 | 多样化风格，快速迭代 |
| 资讯聚合者 | 需要整合多源内容 | 自动采集，智能改写 |

### 1.3 核心价值主张

- **效率提升**: 从采集到发布全流程自动化，节省80%人工时间
- **质量保证**: AI改写保持原创性，多种风格适配不同场景
- **灵活可控**: 支持风格定制、模板选择、人工审核介入

## 3. 功能模块详细需求

### 3.1 账户管理

#### 3.1.1 功能描述
支持多账户管理，每个账户独立配置微信参数和默认风格。

#### 3.1.2 需求列表

| 功能 | 优先级 | 描述 |
|-----|--------|------|
| 创建账户 | P0 | 设置账户名称、ID、微信参数 |
| 编辑账户 | P0 | 修改配置信息 |
| 删除账户 | P1 | 软删除，保留历史数据 |
| 账户切换 | P0 | 快速切换当前操作账户 |
| 统计概览 | P1 | 各账户内容统计 |

#### 3.1.3 数据字段

```python
Account {
  id: string          # 唯一标识
  name: string        # 显示名称
  wechat_appid: string
  wechat_secret: string
  wechat_author: string
  default_style: string    # 默认改写风格
  default_template: string  # 默认发布模板
  created_at: datetime
}
```

### 3.2 灵感库（内容采集）

#### 3.2.1 功能描述
采集外部文章内容，进行AI评分，决定是否进入改写流程。

#### 3.2.2 需求列表

| 功能 | 优先级 | 描述 |
|-----|--------|------|
| URL采集 | P0 | 输入文章URL自动抓取 |
| AI评分 | P0 | 评估爆款潜力、内容质量 |
| 灵感审核 | P1 | 采纳/拒绝/跳过 |
| 批量导入 | P2 | 支持Excel/CSV批量导入 |
| RSS订阅 | P2 | 自动监控RSS源更新 |

#### 3.2.3 AI评分维度

- **爆款潜力** (0-100): 成为爆文的可能性
- **内容质量** (0-100): 信息价值和准确性
- **传播价值** (0-100): 值得分享的程度
- **AI推荐理由**: 50字以内
- **建议改写方向**: 具体的改写建议

#### 3.2.4 状态流转

```
待分析 → 分析中 → 待决策
              ↓
         已采纳 → 已进入流水线
              ↓
         已跳过
```

### 3.3 AI改写引擎

#### 3.3.1 功能描述
核心功能，将采集的内容改写成新文章，支持多种风格和灵感引用。

#### 3.3.2 改写风格

**8种预设风格:**

| 风格ID | 名称 | 特点 | temperature |
|--------|------|------|-------------|
| tech_expert | 科技专家 | 深入浅出，专业不晦涩 | 0.7 |
| business_analyst | 商业分析师 | 数据支撑，商业洞察 | 0.7 |
| storyteller | 故事讲述者 | 情感共鸣，故事包装 | 0.8 |
| popular_science | 科普作家 | 通俗易懂，寓教于乐 | 0.7 |
| opinion_leader | 观点领袖 | 犀利观点，引发思考 | 0.75 |
| trend_observer | 趋势观察者 | 敏锐洞察，前瞻分析 | 0.7 |
| practitioner | 实战派 | 干货满满，实操性强 | 0.7 |
| entertainment | 娱乐向 | 轻松有趣，传播性强 | 0.85 |

#### 3.3.3 灵感引用机制

- **相似度计算**: Jaccard相似度算法
- **引用数量**: 最多3篇相关文章
- **引用阈值**: 相似度 > 0.7
- **处理方式**: 借鉴风格，保持原创

#### 3.3.4 自定义指令

支持额外指令来微调改写:
- "多用比喻，通俗易懂"
- "增加数据支撑"
- "语气更正式一些"
- "适合小白阅读"

### 3.4 排版模板系统

#### 3.4.1 功能描述
发布前选择不同的视觉排版风格。

#### 3.4.2 模板列表

| 模板ID | 名称 | 特点 | 适用场景 |
|--------|------|------|----------|
| default | 经典默认 | 公众号标准风格，清晰易读 | 通用 |
| minimal | 极简风格 | 干净简洁，专注内容 | 深度阅读 |
| tech | 科技风格 | 深色主题，渐变色彩 | 科技内容 |
| business | 商务风格 | 专业稳重，正式感 | 商业分析 |

#### 3.4.3 模板扩展

支持自定义CSS注入:
```python
template = DefaultTemplate(custom_css="""
  .article-title { color: #ff0000; }
""")
```

### 3.5 发布系统

#### 3.5.1 功能描述
将改写排版后的内容发布到微信公众号草稿箱。

#### 3.5.2 发布流程

```
1. 选择文章
2. 选择排版模板
3. 渲染HTML
4. 上传图片素材
5. 创建微信草稿
6. 返回草稿ID
```

#### 3.5.3 发布配置

- 支持定时发布(未来)
- 支持封面图自动选择
- 支持原文链接保留
- 支持作者信息设置

---

### 3.6 流水线（自动化）

#### 3.6.1 功能描述
一键运行完整流程：采集 → 改写 → 发布。

#### 3.6.2 流水线配置

```python
PipelineConfig {
  batch_size: int          # 每批处理数量 (默认3)
  auto_rewrite: bool       # 自动改写
  auto_publish: bool       # 自动发布
  style: string           # 默认改写风格
  template: string        # 默认发布模板
  min_ai_score: float     # 最低AI评分要求
}
```

#### 3.6.3 状态追踪

每篇文章的状态:
- 🧲 待改写
- ✍️ 改写中
- 🧾 待审核
- 🚀 待发布
- 📤 发布中
- ✅ 已发布
- ❌ 改写失败
- ❌ 发布失败

## 4. 用户界面

### 4.1 CLI界面

```bash
# 账户管理
autoplatform account create --name "主账号" --id main
autoplatform account list
autoplatform account switch main

# 灵感库
autoplatform collect --url "https://..."
autoplatform inspirations list
autoplatform inspirations approve <id>

# 改写
autoplatform styles                    # 列出风格
autoplatform rewrite --id <id> --style business_analyst

# 发布
autoplatform templates                 # 列出模板
autoplatform publish --id <id> --template tech

# 流水线
autoplatform pipeline --account main --batch 5

# 统计
autoplatform stats --account main
```

### 4.2 API接口

详见接口文档 (可单独输出 API.md)

核心端点:
- `POST /api/accounts`
- `POST /api/inspirations`
- `POST /api/articles/{id}/rewrite`
- `POST /api/articles/{id}/publish`
- `POST /api/pipeline/process`

## 5. 非功能需求

### 5.1 性能要求

| 指标 | 要求 |
|-----|------|
| 文章采集 | < 10s |
| AI改写 | < 30s |
| 发布到微信 | < 15s |
| API响应 | < 200ms |

### 5.2 数据存储

- **数据库**: SQLite，单文件存储
- **图片存储**: 本地文件系统
- **备份**: 支持导出JSON/CSV

### 5.3 安全性

- API密钥本地存储，不上传云端
- 微信token本地缓存，定期刷新
- 不支持多用户，单机使用

### 5.4 可扩展性

- 风格预设可自定义添加
- 模板系统可扩展新模板
- 支持接入多个AI模型
