# 微信公众号全自动发布系统

> **一键发布**: 提供文章 URL → 自动完成全流程 → 返回草稿箱链接

---

## 🎯 功能

- ✅ 内容获取（标题、作者、正文、图片）
- ✅ 飞书保存（包含所有图片）
- ✅ AI 改写（原创度>60%）
- ✅ 内容审查（敏感词检测）
- ✅ 微信发布（修复中文乱码）
- ✅ 图片处理（自动下载、上传、插入）

---

## 🚀 使用

**只需一句话**:
```
发布这篇文章：https://mp.weixin.qq.com/s/xxx
```

```

   审查报告 + 审查结论
   ↓
开发 (微信发布)
   ↓
   草稿 ID + 草稿箱链接
```
